A simple app to clone (Read & Write EEPROM data), or set a new Odometer value. V2 now supports ZE0 and AZE0 clusters. Expect approx +/-15km difference in the Odo value you requested, and what is shown on the Odo (Rounding errors in my math). Reading/Writing EEPROM data will clone the exact values.

Requires a J2534 passthrough device (Connect to your obd2 port).

I recommend an aliexpress Godiag GD101 or VNCI Nano passthrough.
